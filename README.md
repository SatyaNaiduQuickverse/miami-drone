# Miami Police Drone Management System

A comprehensive drone fleet management system for law enforcement operations.

## Features

- Real-time drone tracking and telemetry
- Mission planning with waypoint navigation
- Live camera feeds (front and bottom cameras)
- Robotic arm control with 3D visualization
- Fleet-wide drone status monitoring
- Dark theme optimized UI

## Tech Stack

- **Frontend**: React, Vite, Leaflet Maps, React Three Fiber
- **Backend**: Node.js, Express, MongoDB
- **Gateway**: Real-time communication layer

## Getting Started

```bash
# Install dependencies
cd client && npm install
cd ../server && npm install
cd ../gateway && npm install

# Start the application
./start.sh
```
